<h2>요구사항</h2>
<h3>회원 관리</h3>

회원 가입, 로그인, 프로필 관리 기능 제공<br>

<h3>게시물 관리</h3><br>

게시물 작성, 조회, 수정, 삭제 기능<br>

게시물에 대한 좋아요 기능 제공<br>

좋아요 수 집계 기능<br>

<h3>댓글 시스템</h3>

게시물에 댓글 작성, 조회, 수정, 삭제 기능<br>

댓글에 대한 대댓글 지원<br>

댓글에 대한 좋아요 기능 제공<br>

좋아요 수 집계 기능<br>

<h3>좋아요 시스템</h3>

게시물과 댓글에 대한 좋아요 기능 통합 관리<br>

사용자별 좋아요 이력 관리<br>

엔티티 타입에 따른 좋아요 구분(POST, COMMENT)<br>

<h3>알림 시스템</h3>

특정 이벤트 발생 시 회원에게 알림 전송<br>
